# Task Manager (CLI)

Aplicación de gestión de tareas desde la terminal, desarrollada en Python.

## Descripción

Task Manager es una aplicación de consola que permite:

- Crear tareas
- Marcar tareas como completadas
- Eliminar tareas
- Guardar la información en un archivo JSON para mantener los datos entre ejecuciones

## Tecnologías utilizadas

- Python 3
- Manejo de archivos JSON
- Librerías estándar (`datetime`, `json`, `os`)

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

git clone https://github.com/santifauro/task-manager.git

2. Entrar a la carpeta del proyecto:

cd task-manager

3. Ejecutar el programa:

python task_manager.py
